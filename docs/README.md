# Documentation

## Structure

| Directory | Purpose |
|-----------|---------|
| `architecture/` | System architecture diagrams and technical overviews |
| `decisions/` | Architecture Decision Records (ADRs) |
| `prds/` | Product Requirements Documents |
| `backlog/` | Feature backlog with priority tracking |
| `research/` | Research notes, spikes, and explorations |
| `archive/` | Superseded or deprecated documents |

## Naming Conventions

- **ADRs**: `ADR-NNN-short-description.md` (e.g., `ADR-001-monorepo-and-tooling.md`)
- **PRDs**: `kebab-case-descriptive-name.md` (e.g., `mvp-community-os.md`)
- **Architecture**: `kebab-case-descriptive-name.md`
- **Backlog items**: tracked in backlog CLAUDE.md, not individual files

## Templates

Each directory contains a `CLAUDE.md` with the template for that document type.
